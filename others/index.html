# Tips

# Office Tool Plus shortcut keys

The new shortcut keys only available in V9.0.2 or later.

  • Esc: Close right panel (if opened).
  • F1: Help.
  • Ctrl + 1: Switch to home page.
  • Ctrl + 2: Switch to deploy page.
  • Ctrl + 3: Switch to activate page.
  • Ctrl + 4: Switch to toolbox page.
  • Ctrl + 5: Switch to documents converter.
  • Ctrl + T: Open setting page.
  • Ctrl + B: Open about page.
  • Ctrl + Shift + P: Open command box.

# Deploy page

  • F5 & Ctrl + R: Refresh Office information and reset deploy settings.
  • Ctrl + O: Open a configuration file.
  • Ctrl + S: Save configuration to local.
  • Ctrl + I: Import configuration from web.
  • Ctrl + L: Install iSlide.
  • Ctrl + Alt + P: Open products list.
  • Ctrl + Alt + L: Open languages list.
  • Ctrl + Alt + D: Ignore errors or warnings, force start deploy.

# Activate page

  • F5 & Ctrl + R: Reload Office information.
  • Ctrl + D: View activation information.
  • Ctrl + E: Clear operation result.
  • Ctrl + Alt + P: Open licenses list.
